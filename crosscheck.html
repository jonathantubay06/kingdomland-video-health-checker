Setup: How to enable auto-update for the spreadsheet

To allow this tool to automatically update your Google Spreadsheet, deploy a small Google Apps Script:

  1. Open your spreadsheet in Google Sheets
  2. Go to Extensions → Apps Script
  3. Delete any existing code and paste the script from google-apps-script.js in this project
  4. Click Deploy → New deployment
  5. Select type: Web app
  6. Set "Execute as" to Me and "Who has access" to Anyone
  7. Click Deploy and copy the web app URL
  8. Add it to your .env file as GSHEET_WEBAPP_URL=your-url-here
  9. Restart the server
